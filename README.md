# Spring Boot Docs + Code Generator (SpringbootAI / DevJAI)

Generate complete documentation and runnable Spring Boot code
from user stories or existing repositories.

---

## What this repository is

This is an end-to-end demo repository that shows the full workflow:

1) User stories as input
2) Generated Spring Boot project from those stories
3) Generated documentation + UML from the generated code

No real customer code or business logic is included.
This repository exists for demonstration and evaluation purposes only.

---

## Repository structure

- `user-stories/` — input user stories
- `generated-code/` — generated Spring Boot projects
- `generated-docs/` — generated documentation (Markdown / PDF / UML)
- `video/` — demo walkthrough
- `screenshots/` — visual evidence
